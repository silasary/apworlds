# Manual_RiftoftheNecrodancer_Axxroy

Built on the Manual APWorld, this allows for any rhythm game to be randomized.
Based on DeamonHunter&#x27;s Muse Dash implementation, play each song you have unlocked in order to get checks!
Collect your Goal Locking items in order to beat your final song!
