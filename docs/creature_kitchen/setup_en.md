# Creature Kitchen Archipelago Setup Guide

## Required Software

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases) (latest release)
- Creature Kitchen (Steam)
- BepInEx 6 IL2CPP (already installed if using the randomizer mod)
- Creature Kitchen Randomizer mod (with AP client support)

## Installation

1. Install the `creature_kitchen.apworld` file into your Archipelago installation using the Archipelago Launcher's "Install APWorld" button.
2. Install the Creature Kitchen Randomizer mod into `BepInEx/plugins/` in your game directory.

## Generating a Game

1. Create a YAML configuration file for your Creature Kitchen settings (see the Options page).
2. Place the YAML in the Archipelago `Players/` folder.
3. Generate the multiworld using the Archipelago Launcher.

## Connecting

1. Launch Creature Kitchen with the mod installed.
2. On the main menu, enter the AP server address, port, slot name, and password.
3. Start a new game — items will be sent and received through the AP server.

## Game Info

Creature Kitchen has 82 randomized locations and 82 items:

- **10 Keys** — Open locked rooms throughout the house
- **2 Crest Halves** — Required to complete the game
- **39 Recipe Cards** — Teach you how to cook dishes for the creatures
- **31 Ingredients** — Finite cooking supplies and mushrooms scattered around the house

Feed all 9 creatures and collect both crest halves to achieve your goal!
