# Pokémon Trozei / Link

**NOTE** currently only the European version under the name Pokémon Link is supported.
Support for the USA and/or Japanese version requires verification of necessary memory addresses.


## Where is the options page?

There is no options page.
You can generate a default YAML by opening `Archipelago Launcher`, and clicking
`Generate Template Options`. This then opens a folder where the YAML can be found
as `Pokemon Trozei.yaml`

## What does randomization do to this game?

This randomizes the order in which levels in adventure mode are played.

## What items and locations get randomized?

Locations are defined as the levels in adventure mode. 
Each level has 2 completion locations.

As this game lacks traditional items, the main items are
level unlocks. Each level has an unlock item.
Besides this currently only filler coin items exist.

## Death link

Sending a death link is triggered by a game over in game
Receiving a death link is buffered until actively linking,.
Only one death link can be buffered at a time.

## What other changes are made to the game?

Unlocking the top right and top left areas requires beating Phobos Mobile No. 1: Phobos Train.<br>
Unlocking the bottom left area requires beating Phobos Mobile No. 3: Phobos Drill.<br>
Unlocking Phobos Secret Fort: Phobosphere requires its unlock item, and having beat 1 to 5 bosses depending on your YAML setting for `required_bosses`<br>
