# Super Mario Sunshine Arcade 2

Created by Augs SMSHacks, Super Mario Sunshine Arcade 2 is a Romhack for Super Mario Sunshine. Featuring 96 platforming gauntlets
based on mechanics and locations from the base game, the player is given the chance to experience Sunshine&#x27;s mechanics in a whole
new light.
