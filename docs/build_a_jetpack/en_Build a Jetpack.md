# Build a Jetpack

## Where is the options page?

The [player options page for Build a Jetpack](../player-options)
contains all the options you need to configure and export a config file.

## What is the goal of Build a Jetpack when randomized?

Reach the star high in the sky. This requires receiving three Jetpack items.

## Which items can be in another player's world?

Every upgrade (diving equipment, raft expansions, etc.) is shuffled into the item pool and can be found as an item.

## What is considered a location check in Build a Jetpack?

Buying upgrades sends location checks. If the option is enabled, selling unique types of fish for the first time will also send location checks.

## What does another world's item look like in Build a Jetpack?

Currently, items are not listed until they are sent. This may change in the future.
