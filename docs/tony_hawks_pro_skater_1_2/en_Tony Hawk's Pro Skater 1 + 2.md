# Tony Hawk's Pro Skater 1 + 2

## Randomizer Summary

Progression has been decoupled from standard Skate Tour advancement and distributed in the multiworld. Instead of unlocking levels by completing a set number of goals and playing with whichever skater you prefer, you must find specific Level Unlocks and Skater Unlocks to progress towards your goal.

Players begin with 3 starting levels and 1 skater. The total number of included levels and skaters is configurable. All skaters begin with severely limited capabilities: They start with 1 random trick type (e.g. Grabs, Grinds etc.) unlocked and all other abilities must be received as items. This means that at any given time, each skater will have varying capabilities based on the items you have received for them. Each skater has their own unique set of location checks.

Progression is both horizontal and vertical. As you receive more Level Unlocks and Skater Unlocks, your pool of available location checks grows. However, your ability to clear those checks depends on having capable enough skaters, creating a dynamic where you must decide which level / skater combination to prioritize for efficient progression.

Goals are tied to Secret Tapes, acting as the "macguffin" items for this game. The player must receive a configurable amount to either win or help gain access to their determined goal level, where they must score 1 million points to win.

**Locations**
- High / Pro / Sick Scores. Per Level
- Platinum Scores. Optional
- High Combos. Per Level
- Pro / Sick Combos. Added by the mod. Per Level
- Platinum Combos. Added by the mod. Per Level
- Collecting each of the S-K-A-T-E letters. Per Level (that has them)
- Collecting the Secret Tape. Per Level (that has them)
- Landing Grinds / Lips / Manuals of a certain duration. Optional. Per Level. Duration will be random per trick type / level
- Landing random gaps. Optional. Per Level. Can customize how many gap locations per level. Only unlocks when going back to the main menu (game limitation)
- Landing each skater's signature specials. Can be done anywhere once the special meter is unlocked. Only unlocks when going back to the main menu (game limitation)

**Items**
- Level unlocks
- Skater unlocks
- Progressive Stats. Per Skater. 4 Stages: Nerfed (initial), Normal, Maxed and Bail Immunity
- Flip Tricks. Per Skater. Flip tricks won't contribute to scoring until received
- Grab Tricks. Per Skater. Grab tricks won't contribute to scoring until received
- Progressive Grind Tricks. Per Skater. 3 Stages: No contribution to scoring (initial), Normal, Perfect Balance
- Progressive Lip Tricks. Per Skater. 3 Stages: No contribution to scoring (initial), Normal, Perfect Balance
- Progressive Manual Tricks. Per Skater. 3 Stages: Cannot initiate (initial), Normal, Perfect Balance
- Progressive Special Meter. Per Skater. 3 Stages: Cannot accrue (initial), Normal, Doesn't reset on bail
- Spin Tricks. Per Skater. Spins won't contribute to scoring until received
- Transfers. Per Skater. Won't be able to perform transfers until received
- Wallplants. Per Skater. Won't be able to perform wallplants until received
- Extra Tricks. Per Skater. Won't be able to perform trick variants (AKA double-tap tricks) until received. Reverts are also considered to be extra tricks
- Stance Switching. Per Skater. Won't be able to switch stances, including nollie position, until received
- Double Score. Per Skater. The base score of every trick is doubled when received


## Randomizer Features
- All 19 levels + 23 Skaters supported
- Ability to select individual levels to include / exclude from generation
- Ability to select individual skaters to include / exclude from generation
- Ability to customize how many levels (8-19) / skaters (1-10) to play
- Ability to set / randomize difficulty of target scores / combo scores. By level or globally
- Each skater has their own ability progression and set of location checks
- 14 unique traps to experience!
- Scales well for all types of multiworlds: Short / Long Syncs, Asyncs; ~75-3250 checks
- Ultra-rare variants of bobblehead filler items (that do nothing) to collect and brag about
- Custom client UI that displays important information and updates in real-time
- No modding! The Archipelago client mods the game in real-time. No need to download / install / configure anything else
- Feature-complete and expected to be very stable


## Tracking

**Universal Tracker**

The APWorld is fully compatible with Universal Tracker. It is a YAML-free implementation (i.e. you don't need to have the YAML used to generate in your Players directory). All score, combo score, long trick and gaps targets are appended to location names.
