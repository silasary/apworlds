# Changelog
Versions are sorted in ascending order, i.e. the most recent changes are at the top.

## 1.0.2

- Fixed goal potentially not being sent if deconnected from the server at least once

## 1.0.1

- Fixed goal only being sent after reconnecting the client to the server
- Fixed some capture quests' names due to bad grammar

## 1.0.0

- Capture quests (off-missions, missions-only, off-mission legendaries, special missions)
- Poké Assist quests
- Item descriptions
- `Complete quest #<1-1000>` and `Inspect a pokémon` as locations
- Multiple unittests for different options
- Options:
  - Quests Count (1-1000, default 20)
  - Goal Amount (1-1000, default 20)
  - Modify Quest Pool
  - Duplicate Quests (default false)
  - Captures Blacklist
  - Quest Plando
  - Force One Check at a Time
- Universal language support
- Docs: Game info page, setup guide, changelog
