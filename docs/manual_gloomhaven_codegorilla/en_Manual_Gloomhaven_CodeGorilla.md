# Gloomhaven (Manual)

## What is a Manual game?
A Manual game is a custom game that is played without full connection from game to Archipelago server. Instead, you use the Manual Client to manually indicate which locations you've checked, driven by the honor system.

## Where is the settings page?
The player settings page for this game is located <a href="../player-settings">here</a>. It contains all the options
you need to configure and export a config file.

## What does randomization do to this game?
Scenarios are locked behind region unlock items; for example, scenarios located in Gloomhaven require the City Adventuring Permit. Story scenarios also require you to complete the normal story progression to reach them, while side scenarios require their region unlock and other non-RNG based requirements. For example, if a scenario normally is unlocked by an event from retiring a particular class, then that class is a requirement, but actually getting the event is not.

The following things are locations that contain multiworld items:
* Scenario completion
* Treasure chests
* Personal quests
* Prosperity thresholds
* Reputation thresholds

Additionally, you can optionally include the Jaws of the Lion DLC and/or the Solo Scenarios DLC.

## What is the goal of Gloomhaven when randomized?
The goal is to get three End of Corruption Achievements, reach the end of the main story quest, and complete Scenario 51.

## Which items can be in another player's world?

The following items are randomized:
* Region Unlocks
* Classes
* Items found in chests/scenario rewards
* Item Designs
* Gold
* Experience
* Checkmarks
* Reputation
* Merchant Upgrades
* Achievements

NOTE: If using the Steam version of the game, the six vanilla starting classes are always unlocked. It's just the honor system to not use any classes you haven't unlocked from those six. When you do unlock one of those classes, create a party member of that class to open any scenarios they're needed to unlock.

## How do I install the mod for Gloomhaven?
If you are playing the digital version of Gloomhaven on Steam, you can use this mod from the Steam Workshop: <a href="https://steamcommunity.com/sharedfiles/filedetails/?id=3158977224">Steam Workshop Mod</a>

Additionally, there is a known bug with modded campaigns that causes save files to become unloadable after exiting the game and restarting it. To work around this, I recommend installing the Gloomhaven Expanded Modding tool (GHEM). You can find it and it's installation instructions here: <a href="https://github.com/Acenm5/GHEM">Gloomhaven Expanded Modding on GitHub</a>
