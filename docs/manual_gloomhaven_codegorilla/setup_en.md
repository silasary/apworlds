# Manual Randomizer Setup Guide

## Required Software (for Steam)

- Archipelago Manual Support on Steam Workshop: <a href="https://steamcommunity.com/sharedfiles/filedetails/?id=3158977224">Steam Workshop</a>
- Gloomhaven Expanded Modding on GitHub: <a href="https://github.com/Acenm5/GHEM">GitHub</a>

## Installation Procedures

1. Follow the instructions under the heading INSTALLATION for Gloomhaven Expanded Modding on the link above
2. Subscribe to Archipelago Manual Support on Steam Workshop using the link above
3. Open Gloomhaven on Steam
4. From the main menu, go to Extras -> Modding
5. Go to Manage Mods and confirm the Archipelago Manual Support mod is there.
6. Go back to the modding menu, and select Custom Rulesets
7. Select Create New Ruleset
8. Select Campaign Ruleset
9. Enter a name for your ruleset, and select the Archipelago Manual Support mod, then select Create
10. Select your new ruleset from the menu
11. Select the compile button
12. When the compile finishes, select Play 
13. Select New Campaign and create your campaign
14. When loading your save file later, you can skip steps 7 through 11 and just go straight to selecting Play on your ruleset. You should then have the option to Load Campaign in addition to New Campaign.

## Joining a MultiWorld Game

Use the Manual Client to connect. See <a href="https://github.com/ManualForArchipelago/Manual/blob/main/docs/play/connect-client.md">the GitHub documentation</a> for more details.
