# YOHANE THE PARHELION -BLAZE in the DEEPBLUE- Setup Guide

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases)

With the game open, open the **YOHANE THE PARHELION -BLAZE in the DEEPBLUE- Client** from the Launcher and connect it to the Archipelago server.
