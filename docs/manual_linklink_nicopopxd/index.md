# Manual_LinkLink_Nicopopxd

Fork of the original Manual_LinkLink_Silasary.
 Do you like item links? Do you wish you could link them together?
 With this you can link your links and play all the links (and more) games together.
 This is also a base that you can customize. Take a peek at the Readme if that interest you.
