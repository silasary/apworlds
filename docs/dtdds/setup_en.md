# Door to Door Door Salesman Randomizer Setup Guide


### Required Software:
- [Archipelago Launcher](https://github.com/ArchipelagoMW/Archipelago/releases/latest)
- [Door to Door Door Salesman Client & APWorld](https://interestedsc2.itch.io/door-to-door-door-salesman)

### Steps:
1. Download the above software
2. After installing the launcher, open the apworld to install it as well
3. Open the "Options Creator" in the launcher
4. Configure and export a yaml file to the players folder
5. Open the "Generate" option in the launcher
6. Upload the generated file to [Archipelago](https://archipelago.gg) and open a new room
7. Open sweet defeat
8. On the menu, go to the "Archipelago" tab on the side
9. Enter the port and slot name
10. Click connect
11. Have fun!
