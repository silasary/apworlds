# Dark Souls II Locations

## Location Groups

### Regions and Sub-Regions

Regions can include subregions, indicated by a ' - ' separating the main region name from its subregion. Excluding a region, such as 'Brume Tower,' will also exclude all of its subregions.

### Item Categories

There exist location groups for all item categories as well. This means that if you exclude the location group "Weapons" all locations that originally had weapons in the unrandomized game will be excluded.
