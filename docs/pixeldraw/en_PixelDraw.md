# PixelDraw

Pixeldraw is a sandbox game that relies upon scanning shapes you find on the ground, gaining tools and different shapes to use.

## Controls

- Pause - Escape
- Move Foreword - W or Up Arrow
- Move Backward -  S or Down Arrow
- Move Left - A or Left Arrow
- Jump - Space
- Scan - Left Click
- Use Tool - Right Click
- Switch Tools - Scroll or numbers 1 through 0
- Return - R
- Unstuck - U
