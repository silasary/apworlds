# Supraland

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a
config file.

## What does randomization do to this game?

All items in chests are randomized, as well as those in shops.
Coinsanity and Gravesanity adds those in as items and locations.

## What Terraria items can appear in other players' worlds?

All items normally found ingame, and floating coins/graves if selected.

## What does another world's item look like in Terraria?

An Archipelago Icon Disk
