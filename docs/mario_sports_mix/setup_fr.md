# Mario Sports Mix tuto d’installation 

## Logiciels requis 

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases) (la version 0.6.7 ou la plus récente)
- [Dernière version de l’Apworld](https://github.com/ElectroStarz/ArchipelagoMSM/releases)
- [AP Gecko Codes](https://github.com/ElectroStarz/ArchipelagoMSM/releases) (trouvable en téléchargement avec l’Apworld)
- [L’émulateur Dolphin](https://dolphin-emu.org/download/?ref=btn)
- Votre rom légal de Mario Sports Mix Européenne. (La version US sera supporté plus tard dans le développement, la version
JPN possiblement.)
 
## Installation

### Archipelago

1) Téléchargez [la version la plus récente d’Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases)
2) Téléchargez [la version la plus récente de l’Apworld](https://github.com/ElectroStarz/ArchipelagoMSM/releases) et
glissez-la dans votre fichier “Custom_Worlds”. Une méthode plus simple consiste à utiliser la fonction “Install APworld”
directement dans l’application, et sélectionnez le fichier installé.

### Dolphin 

1) En détention de votre fichier rom, mettez là dans votre fichier contenant tous vos roms
2) Allez dans les options de l’application, et dans la catégorie “Général” activez “Activer les cheats”
3) Clique droit sur le jeu Mario Sports Mix et allez sur “Propriété”, ensuite sur “Gecko Codes” et cliquez sur “Ajouter
un nouveau code” Une interface s’ouvrira avec des informations à rentrer. Nommez le code “AP Codes” dans le nom du code,
mettez ensuite dans “Créateur” Saulf, enfin dans la partie “Code” vous copiez le code trouvable dans “AP Gecko Codes.txt”
précédemment installé avec l’APworld.

## Notes à lire 

### Archipelago 

- Vérifiez que vous utilisez le client Mario Sports Mix et que vous êtes connecté quand vous jouez au jeu, sans quoi,
**ça ne fonctionnera pas.**

### Dolphin

- <ins>**Sauvegardez bien vos données lorsque vous jouez!**</ins>
- **Assurez vous que vous avez bien mis en place les Gecko code, sans quoi, ça ne marchera pas.**
- Quand vous jouez, n’utilisez que la sauvegarde 1 ! (Le support des autres fichiers de sauvegardes sera possiblement
supporté plus tard.)

## Créer le fichier yaml
### Explication d’un fichier yaml et pourquoi en avoir besoin

Le fichier yaml est ce qui permet à Archipelago comment il doit générer votre jeu. Il regarde vos différentes options et construit la “Logique” du multi-monde grâce à ça.

### Comment le créer

Deux moyens sont à votre disposition pour le créer. La première est celle recommandée, la seconde quand à elle, est celle ceux souhaitant modifier le fichier par eux même.

1ère solution : dans votre application Archipelago, Ouvrez “Option Creator”. Il contient tous vos Apworld natifs à
l’application ou ajoutés, cherchez Mario Sports Mix à gauche et choisissez vos options !

2ème solution : toujours dans Archipelago, ouvrez “Generate Template Options”, trouvez Mario Sports Mix. Ouvrez-le et
modifiez vos options via votre application de bloc-note. N’oubliez pas de sauvegarder le fichier autre part.
