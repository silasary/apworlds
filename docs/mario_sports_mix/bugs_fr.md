# Avertissement

Il est possible que vous ne puissiez pas obtenir le nombre de personnages requis dans le menu de sélection. Utilisez
dans ce cas la fonction « Aléatoire ». Cependant, si vous utilisez cette fonction alors que vous n'avez qu'un seul
personnage de disponible et que vous l'avez sélectionner, le jeu risque de planter. Si cela se produit, utilisez la
fonction aléatoire lorsque vous êtes sur la case Mii.
