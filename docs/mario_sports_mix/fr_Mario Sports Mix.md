# Mario Sports Mix document d’information

## Quel est l'impact de la randomisation sur le jeu ?
La randomisation mélange l'intégralité des personnages et leurs costumes*, les sports, les terrains, les tournois, les
coups spéciaux ainsi que la répartition des objets dans les cases « ? ».

Pour participer à un tournoi, deux conditions doivent être remplies : avoir débloqué l'accès au tournoi et posséder les
terrains associés. Ces derniers sont universels, quel que soit le sport pratiqué. Par exemple, pour disputer la Coupe
Champignon, vous devez impérativement disposer des trois terrains requis (dans l'ordre : Stade Mario, Plage Koopa et
Dock DK). Notez que si vous possédez uniquement le premier terrain, vous pouvez entamer le premier round, mais les
suivants resteront verrouillés jusqu'à ce que vous ayez débloqué le terrain correspondant.

Concernant le mode Sports Mix, deux options permettent de le débloquer selon vos options :
Débloquer l’élément « Sport : Sports Mix ».
Obtenir les 4 cristaux de sport.

* Note : les costumes ne permettent pas d'incarner le personnage correspondant. Si vous possédez un costume de Yoshi,
mais que vous n'avez pas débloqué le personnage Yoshi lui-même, vous ne pourrez pas l'utiliser.

## Quel est l’objectif final ?

Le jeu propose deux boss : Béhémoth et le Roi Béhémoth. Selon vos objectifs, vous devrez en affronter un des deux. 
Voici comment les débloquer :
Béhémoth : obtenir toutes les coupes dans tous les sports. Vous l'affronterez après avoir remporté la finale de la Coupe
Étoile de votre dernier sport.
Roi Béhémoth : obtenir toutes les coupes dans le mode Sports Mix. Comme pour Béhémoth, vous l'affronterez après la finale
de la Coupe Étoile.

## Quels sont les checks du jeu ?

Chaque victoire dans un match de tournoi vous rapportera un check (Exemple : gagner le 2e match de la Coupe Champignon
en difficulté facile/difficile). 
Remporter une coupe.
Débloquer un personnage.
Gagner sur chaque terrain en match amical, dans les difficultés choisi dans les options (Exemple : battre l'IA en
difficulté expert sur le terrain Stade Mario à la balle au prisonnier)

Étant aux premiers stades du développement d'Archipelago pour Mario Sports Mix, de nombreuses options ajoutant des
checks sont prévues à l'avenir, tel que :

- Charactersanity
- Scoresanity
- Stagesanity (terrains de jeu)
- Les chemins annexes des tournois
- Les mini-jeux

## Quels objets peut-on trouver dans les autres mondes ?

- L'accès aux différents sports (une option ultérieure permettra de débloquer nativement tous les sports.)
- Les personnages
- Les différents accès aux coupes dans chacun des sports (Avec la différenciation, entre facile et difficile sauf pour le
Sports Mix mode.)
- Le coup spécial de chaque personnage
- Les objets de la case "?"
- Les pièges (Actuellement : suppression de pièces, gel des personnages et division par deux du temps de match actuel)

Les items pour le jeu sont pour la plupart assez flous sur comment les nommer pour trouver un indice. De ce fait, un tag
a été mis en place pour mieux se repérer. Vous pouvez retrouver un exemple pour chacun d’entre eux : 

- Sport: Sports Mix
- Basketball: Mushroom Cup (Hard)
- Dodgeball: Flower Cup (Normal)
- Volleyball: Star Cup (Normal)
- Hockey: Star Cup (Hard)
- Sports Mix: Star Cup
- Sports Crystal: Red
- Stage: Star Ship | Stage: Behemoth Stage
- Character: Luigi
- Costume: Tennis-wear Peach
- ? Panel: Green Shell
