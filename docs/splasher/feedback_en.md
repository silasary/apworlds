# How to provide Feedback

As this world is in early development, you may encounter issues and it will be helpful to receive relevant feedback if it happens. As such, here's a suggestion of how to provide useful feedback to help you and the mod's development.

## Information to provide

- **Error Log** : Found in `<Splasher>/BepInEx/LogOutput.log`. [**Required**]
- **Describing the context** : Describe when it happened (upon connection, when playing, upon receiving an item...) and what you did afterwards. [**Important**]
- **Options file** : The file used to generate your game, helps to reproduce the issue [**Important**]
- **Spoiler Log** : Found in the generated game archive in `<Archipelago>/output/<Most recent archive>`. It provides more information about the game, and allows to generate the same game as yours if needed. [**Important**]
- **Room link** : If playing on `archipelago.gg`, you can provide you're room link so I can reproduce the issue more easily. [**Appreciated**]
- **Operating System** : Though the mod should not be OS-dependant, it is not yet crystal clear. [**Appreciated**]

## Where to contact

- DM on Discord (@frisk2892) so we can exchange about what happened
- Post on the Archipelago Server **on the game's thread** (in `future-gale-design`). To avoid flooding the thread, **please refer DMs or create a thread in the discussion**.
