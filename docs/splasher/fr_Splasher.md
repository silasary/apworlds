# Splasher

## En quoi consiste ce jeu ?

**Splasher** est un jeu de plateforme 2D créé par la Splashteam, dans lequel vous contrôlez un prisonnier courageux d'une entreprise douteuse, Inkorp, qui produit de la peinture aux propriétés étonnantes.

## Comment le randomizer impacte le jeu ?

L'idée est de casser la linéarité du jeu pour créer une nouvelle expérience, cela grâce aux modifications (optionnelles) suivantes :
- Vous pouvez débloquer tous les niveaux dès le début de votre partie, ou bien verrouiller ceux-ci derrière des clés réparties dans l'*Archipelago*.
- Vous pouvez verrouiller vos pouvoirs derrière des *items* Archipelago, ce qui demandera occasionnellement une certaine inventivité pour progresser.

Quant aux *locations* (checks), les collectibles principaux du jeu (les *Splashers*) sont mélangés dans l'Archipelago. Il y a aussi la possibilité d'inclure le Contre-la-Montre dans votre partie.

## Quel est l'objectif ?

Comme dans le jeu de base, vous devez emprisonner le scientifique en chef et libérer tous les prisonniers ! Atteindre le dernier niveau demande un nombre personnalisable de Splashers.

## Points instables de cette version

- La logique est instable, même si raisonnablement testée de sorte qu'il est peu probable qu'il reste des problèmes majeurs.
- L'interface n'a pas encore étaient retravaillée pour l'Archipelago, en particulier vous n'aurez pas de messages Archipelago dans votre jeu. Il est très fortement recommandé de jouer avec un tracker.
