# Comment fournir des retours d'expérience

Ce monde est en développement précoce, ainsi il est possible que vous rencontriez des problèmes qu'il est utile de signaler. Pour faciliter les échanges que nous aurons sur ce sujet, voici une liste d'informations que vous pouvez fournir pour expliquer le problème rencontré.

## Informations à fournir

- **Rapport d'erreur** : Trouvable à `<Splasher>/BepInEx/LogOutput.log`. [**Requis**]
- **Description du contexte** : Décrivez quand le problème s'est produit (pendant la connexion, en jeu, en recevant un item...) et ce que vous avez fais ensuite. [**Important**]
- **Options** : Le fichier d'options pour générer votre partie. Cela aide à reproduire votre situation. [**Important**]
- **Fichier Spoiler** : Trouvable dans l'archive de votre partie générée : `<Archipelago>/output/<Most recent archive>`. Cela fournit d'avantage d'informations sur votre partie, et me permet de la recréer pour la tester. [**Important**]
- **Lien de session** : Si vous jouez sur `archipelago.gg`, vous pouvez fournir votre lien de partie pour que je puisse directement me connecter à votre partie. [**Apprécié**]
- **Système d'exploitation** : Le mod est normalement multi-plateforme, mais ce n'est pas encore gravé dans le marbre. [**Apprécié**]
