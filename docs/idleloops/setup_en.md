# Idle Loops Randomizer Setup Guide

## .yaml Configuration

- This game has a restrictive start, middle, and end, where it never really opens up and each progression item only unlocks a handful of locations. As such, any progression balancing below normal has caused generation issues for me, but these are presumably less of a problem for multiworlds.

## Required Software

- [The game](https://lloyd-delacroix.github.io/omsi-loops/)
- The [Tampermonkey](https://www.tampermonkey.net/) Extension to install the mod (Greasemonkey does not work/is not supported)

## Installation
- If you have Tampermonkey, clicking [this link](https://ihavenofunnyname.github.io/IdleLoopsAP/idle-loops-ap.user.js) should prompt you to install the mod.
- If it's installed, the AP connection form should appear at the top right of the game.
