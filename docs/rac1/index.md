# Ratchet &amp; Clank

Ratchet &amp; Clank is a third-person shooter platform video game developed by Insomniac Games
and published by Sony Computer Entertainment for the PlayStation 2 in 2002. It is the first
game in the Ratchet &amp; Clank series and the first game developed by Insomniac to not be owned by Universal
Interactive.
