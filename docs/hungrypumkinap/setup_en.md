## Hungry Pumkin AP Setup

### What you need

- `HungryPumkinAP.apworld`
- `HungryPumkin.yaml`
- Access to an Archipelago installation version `0.6.6` or newer
- The web client at `https://hungrypumkin.com`

### Host setup

1. Install `HungryPumkinAP.apworld` into your Archipelago custom worlds folder.
2. Copy `HungryPumkin.yaml` and edit the player name and any options you want.
3. Generate the multiworld with `Hungry Pumkin` enabled.
4. Host the room normally through Archipelago.
5. Give the player their room address in the form `archipelago.gg:PORT`, along with their slot name and password if one is set.

### Player setup

1. Open `https://hungrypumkin.com`.
2. Click the AP logo in the top-left corner of the game.
3. Enter:
   - the room address, like `archipelago.gg:PORT`
   - your slot name
   - your password, if your room uses one
4. Click `Connect`.

### Notes

- The web client is intended to work like other browser-based Archipelago worlds.
- Website-hosted Archipelago rooms should work by entering the given `archipelago.gg:PORT` address directly.
- If someone hosts their own private Archipelago server, it must support secure browser connections to work from the HTTPS web client.
