# Nuclear Throne

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a config file.

## What does randomization do to this game?

Access to Characters, Weapons, and Mutations are randomized into the multiworld and become unavailable until received via said multiworld. Opening a weapon chest without a viable weapon unlocked gives you nothing.

## What Nuclear Throne items can appear in other players' worlds?
- Any Character aside from your starting selection
    (Fish, Crystal, Eyes, Melting, Plant, Y.V., Steroids, Robot, Chicken, Rebel, Horror, Rogue, Cuz)
- Any Weapon
- Any Mutation except Last Wish
- Confetti, a junk item that spawns confetti around the player

## What is considered a location check in Nuclear Throne?
Performing the following tasks as each of the 13 characters are location checks:
- Defeating every enemy in a level, spawning an exit portal
- Opening a Crown Vault (Only requires 1 per character for the location check)
- Sitting on the Throne (Same location check as defeating the Throne to the next loop)
- Defeating the Captain
- OPTIONAL: Collecting each crown (Either loop or end a run defeating Throne or Captain with a crown equipped)
- OPTIONAL: Unlocking B-skin and C-skin (Opting in for these checks requires a .sav file with all skins locked)

## When the player receives an item, what happens?
A sound effect will play along with a message appearing above the player's head. The player must be actively in a run to receive items and become immediately available to the player.
