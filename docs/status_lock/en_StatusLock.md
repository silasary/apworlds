# Status Lock

Status Lock est un monde méta pour Archipelago où les joueurs collectent des cristaux spéciaux pour débloquer des "statuts" in-game tels que le coût des indices, le release et la collecte.

**Résumé rapide:**
- **Jeu:** Status Lock
- **Objectif:** Générer des objets multiverse débloquant des statuts qui affectent la progression.
