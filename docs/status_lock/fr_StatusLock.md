# Status Lock

Status Lock is a meta-game world for Archipelago where players collect special crystals to unlock in-game "statuses" such as hint cost, release and collect.

**Quick summary:**
- **Game:** Status Lock
- **Purpose:** Generate multiworld items unlocking statuses that affect progression.
