# Yu-Gi-Oh! Dungeon Dice Monsters Location Group Info

## Location Groups

Each subheader listed below covers a list of locations that can be referred to in the yaml when generating your game.

### Duelists

Covers all Duelists that can be encountered in Free Duel (IE Serenity Wheeler, Seto Kaiba, etc)

### Division 1

Covers all Division 1 Tournaments (IE Lunch Selects Cup, World Championship, etc)

### Reverse Division

Covers all Reverse Division Tournaments (IE Dominator's Holy War, Millennium Kingdom Cup, etc)

### Dark Division

Covers all Dark Division Tournaments (IE Dark Ceremony, Coliseum of the Dead, etc)

### Grandpas Shop

Covers all the dice that can be purchased at Grandpas Shop. See [Grandpas Shop Guide](docs\Grandpas%20Shop%20Guide.md)

### Shop Tier 0

Covers all the dice that can be purchased at Grandpas Shop with 0 Shop Progression items. See [Grandpas Shop Guide](docs\Grandpas%20Shop%20Guide.md)

### Shop Tier 1

Covers all the dice that can be purchased at Grandpas Shop with 1 Shop Progression item. See [Grandpas Shop Guide](docs\Grandpas%20Shop%20Guide.md)

### Shop Tier 2

### Shop Tier 3

### Shop Tier 4
