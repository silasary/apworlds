# Backlog Expedition Multiworld Setup Guide

## Web version

### Required Software

- Backlog Expedition [Webpage](https://archipelago-backlogexpedition-game.onrender.com/)
- Backlog Expedition AP World from: [GitHub](https://github.com/Nikkilites/Archipelago-BacklogExpedition-APWorld/releases)

### Installation and Connecting

1. Simply connect to the server on the webpage and log in
2. Enjoy!

## Console version

### Required Software

- Backlog Expedition from: [GitHub](https://github.com/Nikkilites/Archipelago-BacklogExpedition-Game/releases)
  - Only tested on Win 10 & 11
- Backlog Expedition AP World from: [GitHub](https://github.com/Nikkilites/Archipelago-BacklogExpedition-APWorld/releases)

### Installation and Connecting

1. Download the game
2. If you are on Win 11, run the .exe as administrator (The game does not work properly in Powershell, and that is the default of win 11. Please use Command Prompt)
4. Connect to the server within the game
5. Please note that, in this version you navigate with typing only, and it looks the best in full screen!
4. Enjoy!
