# Sonic Colors (DS)

## Where is the options page?

You can read through all the options and generate a YAML [here](../player-options).

## What does randomization do to this game?
Once you boot up the game, the game will be set to a state just before the final boss.
At the beginning only the first area of the game will be available, and you are unable to use any wisps.
Collect Shuttles and Wisp translators to access more planets and eventually fight the final boss.

## What items and locations get randomized?
Wisp abilities and planet access will be randomized, along with Chaos Emeralds if the goal is set to `Nega-Mother Wisp`.
Items are placed at Level Clears above a certain rank and additionally Red Star Rings can be added.

## What does another world's item look like in Sonic Colours (DS)?
As the game is unpatched, items appear unchanged in Sonic Colours (DS).
