# Terra Nil Setup Guide

## Required Software

- The latest version of Terra Nil on PC: [Steam](https://store.steampowered.com/app/1593030/Terra_Nil/) (The GoG and Epic versions of the game have not been tested, however they _should_ work as well)
- [BepInEx](https://docs.bepinex.dev/index.html)
- The latest version of the Terra Nil Archipelago mod: [GitHub Releases](https://github.com/nepcore/TerraNilAP/releases)

## Installing the mod

1. Install the latest version of Terra Nil
2. Install the BepInEx mod loader as described in their [installation guide](https://docs.bepinex.dev/articles/user_guide/installation/index.html)
3. Extract the contents of the `TerraNilAP.zip` from the GitHub release into the `BepInEx/plugins` folder in your game files

# Joining a multiworld

1. Run Terra Nil with BepInEx
   - on Linux/MacOS run the `run_bepinex.sh` script located in the game files
   - on Windows run the game as normal
2. On the profile selection screen select the `Archipelago` profile
3. Enter the connection info and hit enter or press the `Connect` button
4. Select `New Restoration` to start the game
