# Nonopelagram Setup Guide

## Required Software

- A modern web browser (Chrome, Firefox, Edge, Safari)
- Access to the Nonopelagram Archipelago client (web-based)

## Connecting to a Server

1. Open the Nonopelagram client in your web browser
2. In the Archipelago panel, enter:
   - **Host**: The server address (e.g., `archipelago.gg` or `localhost`)
   - **Port**: The server port (default: `38281`)
   - **Slot Name**: Your player name from the YAML
   - **Password**: Room password (if required)
3. Click **Connect**

## Gameplay

### Objective
Complete nonogram puzzles to find location checks and receive items from other players.

### Controls
- **Left Click**: Fill a cell
- **Right Click**: Mark a cell with X (once unlocked)
- **Click + Drag**: Paint multiple cells (once unlocked)

### Items You Can Receive
- **Hint Reveal**: Reveals an additional row/column hint per puzzle
- **Extra Life**: Increases your maximum lives
- **Coin Bundle**: Grants coins for the in-game shop
- **Random Cell Solve**: Solves one random cell

### Locations (Checks)
- **First Line Completed**: Complete any row or column
- **Puzzle Milestones**: Complete 1, 2, 4, 8, 16, 32, and 64 puzzles

### Tips
- Start with smaller puzzles to build up items
- Use the shop to buy temporary hints when stuck
- Watch your lives - mistakes cost lives!
