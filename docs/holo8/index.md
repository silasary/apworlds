# holo8

holo8 is an Exit8-like game based around the Hololive idol group released under the HoloIndie project by frog blend. 
Set in Hololive&#x27;s temporary office, you are tasked with reaching the first floor of an endlessly looping office building while detecting and avoiding anomalies.
