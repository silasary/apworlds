## Death Link Overview

#### Cause

Death Link will trigger when:
- Any amount of soldiers die
  - From being attacked or attacking
- Any amount of citizens die
- A spy is executed
- A citizen dies in a brawl

If multiple deaths happen at once, you attack and 7 soldiers die, Death Link will trigger once.

#### Effect

Death Link will:
- Soldier Pop set to 0
- Citizen Pop set to 1
- Default Job and Citizen set to the job that supplies food.

This is to ensure a death spiral doesn't happen where the game is trying to refill other jobs and no food is being produced and everyone dies.
