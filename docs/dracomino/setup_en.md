# Dracomino Randomizer Setup Guide

## Required Software

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest)
- [Dracomino APWorld](https://github.com/DragonWolfLeo/DracominoAPWorld/releases)
- Dracomino
  - [Window and Linux Builds](https://github.com/DragonWolfLeo/Dracomino/releases)
  - [Web Version](https://dragonwolfleo.github.io/Dracomino/)
  - [Source](https://github.com/DragonWolfLeo/Dracomino) You can run the project in [Godot 4.4.x](https://godotengine.org/download/archive/4.4.1-stable/)

## How to play

- Install Archipelago and dracomino.apworld
- Use Archipelago to generate your options/yaml
- Use Archipelago to generate a multiworld
- Host the resulting zip file either with Archipelago or [the website](https://archipelago.gg/uploads)
- Open the game
- Connect to the Archipelago server
- Play
