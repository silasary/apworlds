# Ape Escape 2 Archipelago - Setup Guide
Go ape and track down 300 cheeky monkeys in this Archipelago implementation of "Ape Escape 2", the sequel to the classic PlayStation platformer.

## Prerequisites
- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest)
- [PCSX2](https://pcsx2.net/)
- [The latest `ae2.apworld` file.](https://github.com/dannybonz/ae2_archipelago/releases/latest)
- An Ape Escape 2 ISO. The European PAL release (SCES-50885) and American NTSC release (SLUS-20685) are both compatible.

## Installing the APWorld
- To install the APWorld, open the Archipelago Launcher and click on "Install APWorld", then select your `ae2.apworld` file.

## Configuring PCSX2
In order to facilitate communication between the Archipelago Launcher and the PCSX2 emulator, you must enable PINE.
- Open the "Tools" dropdown and ensure that "Show Advanced Settings" is enabled.
- Open the "System" dropdown and click "Settings".
- Navigate to the "Advanced" section and ensure that PINE is enabled.
- Ensure that the slot number is set to 28011.
  
## Generating
- To generate a game, follow [the official instructions.](https://archipelago.gg/tutorial/Archipelago/setup/en#generating-a-game)

## Playing
- Open PCSX2 and launch your Ape Escape 2 ISO.
- Select the Ape Escape 2 Client from the Archipelago Launcher.
- Use the client to connect to your Archipelago server.
- After the client has connected to both PCSX2 and the Archipelago server, you can start playing!
  
Ensure that you create a brand new save file each time you start a fresh Archipelago world. If you take a break and come back to the game, ensure you resume from the same save file you have been using.

Please note that item and location names are based on the PAL English translation, not the American release.
