# Setup Guide for Flowersanity in Archipelago

## Required Software

- The game on the itch.io page: https://blastslimey.itch.io/flowersanity

## Optional Software

- Archipelago from the [Archipelago Releases Page](https://github.com/ArchipelagoMW/Archipelago/releases)
- Universal Tracker (check UT's channel in the discord server for more information and instructions)

## Configuring your YAML file

### Where do I get a YAML file?

Type &options into the #aginahbot-spam channel in the discord server for more information

## Joining a MultiWorld Game

Upon opening the game, there is an input box in the top right corner of the game's screen.
Put you connection information into the fields and press `Connect`.
If the game connected successfully, the input box will vanish.
If, at some point, the game looses connection to the multiworld, the input box will appear again.
