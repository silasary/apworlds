# Changelog
Versions are sorted in ascending order, i.e. the most recent changes are at the top.

## 1.1.0

- Removed ~500 potentially inappropriate colors
- Fixed some options not having a description
- Fixed world description on webhost having leftover text from copy&paste

### Compatibility
No backwards compatibility

## 1.0.0

- Two goals:
  - Colors count (+ `Goal Amount` option)
  - Specific colors (+ `Goal Colors` option)
- `Pool size` option
- ~8300 colors (all with names)
- 6 merging calculations:
  - `Average`
  - `Addition`
  - `Modulo Addition`
  - `Subtraction`
  - `Maximum`
  - `Minimum`

### Compatibility
First version, so no compatibility issues
