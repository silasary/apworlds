# Rhythm Doctor Multiworld Setup Guide

## Required Software

- Any desktop version of Rhythm Doctor: [Steam](https://store.steampowered.com/app/774181/Rhythm_Doctor/) or [itch.io](https://fizzd.itch.io/rhythm-doctor-early-access)
- Client plugin: Thunderstore (TODO) or [GitHub](https://github.com/nonperforming/RhythmDoctor.Archipelago/releases/latest)

## Installation

You may install the client plugin using a mod manager such as [Gale](https://kesomannen.com/gale) (recommended), [r2modman](https://github.com/ebkr/r2modmanPlus/releases/latest), or [Thunderstore Mod Manager](https://www.overwolf.com/app/thunderstore-thunderstore_mod_manager). Alternatively, you may install the plugin manually.

### Using a mod manager

Open your mod manager, select the Rhythm Doctor game, and install the RhythmDoctor.Archipelago plugin (TODO thunderstore link). Launch the game through your mod manager to enable the client plugin.

### Manual installation

[Install BepInEx 5.4.21](https://docs.bepinex.dev/articles/user_guide/installation/index.html#installing-bepinex-1), launch the game to setup the required folder structure, then download and unzip the [client plugin and its dependencies](https://github.com/nonperforming/RhythmDoctor.Archipelago/releases/latest) under the 'BepInEx/plugins' folder.

## Connecting

Select the Archipelago option on the Main Menu, select the Archipelago option, and follow the instructions on-screen. Upon a successful connection you will be taken to the Main Ward.
