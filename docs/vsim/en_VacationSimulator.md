# Vacation Simulator

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export 
a config file.

## What does randomization do in this game?

The goal in Vacation Simulator is to collect enough vacation region specific memories to reach the 3 sub regions and 
enough memories overall to unlock the final party. These memories are labeled with which vacation region's sub area they 
progress towards unlocking.

The gates to each vacation region are also locked by default requiring a "Gate Unlock" item, which the player starts with 
one of.

Finally the camera, which is required for some tasks, is also shuffled into the multiworld.
